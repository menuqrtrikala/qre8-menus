# ΦίΛιΠs — Ψηφιακό Μενού (QR Web App)

Ψηφιακό μενού για το café-bar **ΦίΛιΠs (γαστροθίορι, Est. 2023)**.
Ο πελάτης σκανάρει ένα **QR code**, ανοίγει η αρχική οθόνη με το logo, και από εκεί
επιλέγει κατηγορία (Cocktails, Spritz, Καφέδες κ.λπ.) και βλέπει τα προϊόντα.

Είναι **static site** — μόνο HTML, CSS και JavaScript. **Χωρίς backend, χωρίς εγκατάσταση,
χωρίς build.** Ανεβάζεις τα αρχεία σε οποιονδήποτε server και δουλεύει.

---

## 📁 Τα αρχεία

```
filips-menu/
├── index.html      → η εφαρμογή (splash → κατηγορίες → προϊόντα)
├── styles.css      → όλη η εμφάνιση / χρώματα / στυλ
├── app.js          → η λογική (πλοήγηση, εμφάνιση)
├── menu-data.js    → ✅ ΕΔΩ αλλάζεις ΤΙΜΕΣ & ΠΡΟΪΟΝΤΑ
├── qr.html         → φτιάχνεις/κατεβάζεις το QR code
├── assets/         → logo και εικονίδια
└── README.md       → αυτό το αρχείο
```

> **Ο κανόνας:** Για να αλλάξεις το μενού, αγγίζεις **ΜΟΝΟ** το `menu-data.js`.

## 1) 🌐 Πώς ανεβάζω τα αρχεία σε server

Διάλεξε όποιον τρόπο σε βολεύει:

### Α) Απλό shared hosting / FTP (το πιο συνηθισμένο)
1. Συνδέσου στον server σου με ένα πρόγραμμα FTP (π.χ. **FileZilla**) ή από τον
   File Manager του cPanel.
2. Ανέβασε **όλα** τα αρχεία (μαζί με τον φάκελο `assets/`) μέσα στον φάκελο
   του site — συνήθως λέγεται `public_html` ή `www`.
3. Άνοιξε το domain σου στον browser (π.χ. `https://to-domain-mou.gr/`).
   Πρέπει να εμφανιστεί η αρχική οθόνη με το logo. ✅

### Β) Netlify / Vercel (δωρεάν, με drag & drop)
- Netlify: μπες στο [app.netlify.com](https://app.netlify.com) → «Add new site» →
  «Deploy manually» → σύρε ολόκληρο τον φάκελο `filips-menu`.
- Σου δίνουν αμέσως ένα URL (π.χ. `https://filips.netlify.app`).

### Γ) GitHub Pages
- Ανέβασε τα αρχεία σε ένα repository → Settings → Pages → επίλεξε branch → Save.

> ⚠️ Σημαντικό: τα ονόματα των αρχείων είναι **πεζά** (μικρά γράμματα). Μην τα
> μετονομάσεις. Το `index.html` πρέπει να μείνει με αυτό το όνομα για να ανοίγει
> αυτόματα.

## 2) 💶 Πώς αλλάζω τιμές / προϊόντα

Άνοιξε το **`menu-data.js`** με έναν απλό επεξεργαστή κειμένου (Σημειωματάριο /
Notepad, ή καλύτερα το δωρεάν **VS Code**). Στην αρχή του αρχείου υπάρχουν αναλυτικές
οδηγίες. Σύνοψη:

**Αλλαγή τιμής** — βρες το ποτό και άλλαξε μόνο τον αριθμό στο `price`:
```js
{ name: "Latte", desc: "", price: "3.2" }   // ← άλλαξε το "3.2"
```

**Προσθήκη ποτού** — αντίγραψε μια ολόκληρη γραμμή `{ ... }` και κόλλησέ την μέσα
στα `items` της κατηγορίας. Πρόσεξε το **κόμμα** στο τέλος:
```js
{ name: "Νέο ποτό", desc: "περιγραφή", price: "5.0" },
```

**Διαγραφή ποτού** — σβήσε ολόκληρη τη γραμμή `{ ... },`.

**Τιμή ποτήρι / φιάλη** (π.χ. κρασιά) — γράψε την τιμή έτσι:
```js
price: "7.0 / 28.0"
```
Θα εμφανιστεί ως `7.0€ / 28.0€`.

Αφού σώσεις το αρχείο, ανέβασέ το ξανά στον server (μόνο το `menu-data.js`) και
ανανέωσε τη σελίδα στο κινητό.

> 💡 Πιο συχνά λάθη: ξεχασμένο **κόμμα** `,` ή **εισαγωγικά** `"`. Αν χαλάσει κάτι,
> σύγκρινε με τις γραμμές δίπλα — πρέπει να έχουν την ίδια μορφή.

## 3) 🔳 Πώς φτιάχνω / ανανεώνω το QR code

1. Άνοιξε το **`qr.html`** στον browser (τοπικά ή από τον server σου).
2. Γράψε το **URL του μενού** σου (π.χ. `https://to-domain-mou.gr/`).
3. Πάτα **«Δημιουργία QR»**.
4. Σκάναρέ το με ένα κινητό για δοκιμή — πρέπει να ανοίγει το μενού.
5. Πάτα **«Κατέβασμα PNG»** και τύπωσε την εικόνα για το τραπέζι.

> Αν αλλάξεις domain στο μέλλον, ξαναμπαίνεις στο `qr.html`, βάζεις το νέο URL και
> φτιάχνεις νέο QR. Το παλιό τυπωμένο QR θα δείχνει στο παλιό URL.

## 4) 🎨 Πώς αλλάζω χρώματα / λογότυπο

### Χρώματα
Άνοιξε το **`styles.css`**. Στην κορυφή, μέσα στο `:root`, είναι οι βασικές μεταβλητές:
```css
--terracotta: #C8794E;   /* πορτοκαλί/terracotta accent */
--sage:       #8FB08A;   /* φασκόμηλο/πράσινο accent */
```
Άλλαξε τους κωδικούς χρωμάτων (hex) και θα αλλάξουν παντού στο site.

### Λογότυπο
- Το logo στην αρχική σχηματίζεται από κείμενο μέσα στο `index.html` (η λέξη
  `ΦίΛιΠs` με τα χρωματιστά σχήματα). Δεν χρειάζεται εικόνα.
- Υπάρχει και έτοιμο **`assets/logo.svg`** αν θες να το χρησιμοποιήσεις αλλού
  (π.χ. στο QR poster).

### Όνομα / σλόγκαν
Στο `menu-data.js`, στο πεδίο `brand`, μπορείς να αλλάξεις όνομα, tagline και τα
σλόγκαν που εμφανίζονται στο footer.

## 🌗 Έξτρα

- **Dark mode:** ενεργοποιείται **αυτόματα** αν το κινητό είναι σε σκούρο θέμα.
- **Γλώσσα GR/EN:** πάνω δεξιά υπάρχει διακόπτης για τα στοιχεία του UI.
- **Back button:** το κουμπί «πίσω» του κινητού δουλεύει σωστά (μέσω URL hash).
- **Χωρίς internet για fonts:** αν δεν φορτώσουν οι Google Fonts, χρησιμοποιούνται
  οι γραμματοσειρές του συστήματος.
